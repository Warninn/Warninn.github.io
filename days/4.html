Some HTML tags that are helpful

<center>

This tag is used to center align the content within its parent element.

Warning: This tag is not supported in all browsers and that's why it may be colored red when you type it.


<marquee>

this tag is used to create scrolling text.

Warning: <marquee> elements must not be present because they are deprecated, increase difficulty for users with limited dexterity, and are distracting for users with cognitive or attention deficits.


EXAMPLE
<div>

this tag divides up code so your code can look clean

It does nothing unless css or JavaScript is added to it.

good code does not overuse divs but does not underuse divs. think of it like groups of code that look good in these bunches


<a>

a stands for anchor link

this is a link to another page or website

href is the element you pass in to where you want to go and target is the way you want to open the link, in a new tab with _blank or _self in the same tab

Example

CODE:
<a href="https://www.google.com" target="_blank">Google</a>
Result:
Google
